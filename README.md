
# Sorting Visualizer 🧠📊

A modern, responsive Sorting Visualizer built using **HTML**, **CSS**, and **JavaScript**. This project visually demonstrates how various sorting algorithms work step-by-step, helping users understand the core concepts through animation.

---

## ✨ Features

- ⚡ Interactive controls to change **array size** and **speed**
- 📈 Visual representation of popular **sorting algorithms**
- 📱 Responsive design (works on desktops & tablets)

---

## 🔍 Algorithms Implemented

-  Bubble Sort
-  Selection Sort
-  Insertion Sort
-  Merge Sort
-  Quick Sort
-  Heap Sort

---

## 🎮 How to Use

1. **Set the array size** and **animation speed** using the sliders.
2. Click **“Generate New Array”** to create a fresh dataset.
3. Click any sorting algorithm button to see it in action!
